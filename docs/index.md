# SmartDeal Hunter

Privacy-first Chrome extension that scores Amazon products on **True Value** and **Personal Fit** using on-device heuristics. Zero remote analysis. No tracking. No affiliate links.

- Source: [github.com/galaxycoils/smartdeal-hunter](https://github.com/galaxycoils/smartdeal-hunter)
- Latest release: [v0.2.1](https://github.com/galaxycoils/smartdeal-hunter/releases/latest)

## Privacy summary

- All product analysis runs **on your device**.
- Review text, browsing history, and personal data **never leave your browser**.
- The only optional remote call is "Deep Check" (rate-limited, opt-in, off by default).
- No affiliate links. No analytics. No tracking pixels.

Read the full [Privacy Policy](./privacy-policy.html) for details.
